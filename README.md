# Lum Ko Sand Portfolio

Personal portfolio site for Lum Ko Sand.

## Run locally

```bash
git clone <your-repo-url>
cd dnasok.github.io
copy env.example .env
npm install
npm start
```

## Test

```bash
npm test -- --watchAll=false
```

## Build

```bash
npm run build
```

## Customize

Edit [src/portfolio.js](src/portfolio.js) for your content and [public/index.html](public/index.html) for SEO metadata.

If you want GitHub or Medium data to load locally, set the values in [.env](.env) from [env.example](env.example).
